#### Como utilizar

```sh
# Clone este repositório
$ git clone https://github.com/rafaelspereira1/Frexco-DevOps

# Acesse o diretório do projeto
$ cd Frexco-DevOps

# Inicialize o container com o docker compose
$ docker-compose up

# Para finalizar o container
$ docker-compose down

```
